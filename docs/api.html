crispy.core.pvplot(fraglist: list[Fragment], xlim: list[int] | None = None, ylim: list[int] | None = None, zlim: list[int] | None = None, show_node: bool = True, show_bond: bool = True, show_axe: bool = False, auto_close: bool = False, camera_position: list[tuple] | None = None, save: bool = True, filename: str = 'plot', save_dir: str = '/home/lbremaud/Bureau/GIT/CRISPY/crispy_git/docs-source/img') → None[source]

Plot fragments.

Parameters:

  • fraglist (list[Fragment]) – List of Fragment objects.

  • xlim (list[int] | None, optional) – Set plot x-limits. The default is None.

  • ylim (list[int] | None, optional) – Set plot y-limits. The default is None.

  • zlim (list[int] | None, optional) – Set plot z-limits. The default is None.

  • show_node (bool, optional) – Show nodes. The default is True.

  • show_bond (bool, optional) – Show bonds. The default is True.

  • show_axe (bool, optional) – Show axes. The default is False.

  • auto_close (bool, optional) – Automatically close window plot. The default is False.

  • camera_position (list[tuple] | None, optional) – Set the camera position (see https://docs.pyvista.org/api/plotting/_autosummary/pyvista.cameraposition#pyvista.CameraPosition). The default is ‘xy’.

  • save (bool, optional) – Save option. The default is True.

  • filename (str) – Filename of saved image.

  • save_dir (str, optional) – Save directory. The default is “/img” in current directory.
